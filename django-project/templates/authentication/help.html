Sign Up

1. You can either click get started on the landing page or you can press the "sign up" hyperlink on the login page to get to the sign up page.

2. From here you write your first and last name in the first two forms. You can not leave these blank.

3. Now write your username, this only accepts alphanmumeric characters.

4. Place your email in the following input form, you must have an @ to work.

5. Write your password, it cannot be nothing and be sure not to forget it.

6. Press create account, this will create your account and re-direct you to the login.

|

Log In

1. In order to log in you can first press the button on the top right of the landing page titled sign in. Alternatively you can navigate to the /signin off of the root browser path.

2. Now you want to place your username inside of the username input field on the right, this only accepts alphanumeric characters.

3. Following this type your password into the password input box and hit log in.

4. You should now be redirected to the dashboard to which you can start generating.

|

Generate Report

1. First you want to navigate to the dashboard, you must be logged in to do this, it will appear on the dashboard after signed in.

2. On the dashboard write the corrosponding class and student name under the first two labels.

3. Adjust each of the sliders until you have the desired setting.

4. Press generate and after a second you can either generate again or copy the text and save it if you like it.

|

Log Out

1. You can log out on the dashboard via the logout icon next to your name or on the landing page.

2. You can also hit the url path signout, this should work as well.
